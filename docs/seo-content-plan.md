## On-Page SEO Pflicht-Checks pro Artikel

- [ ] H1 enthält Hauptkeyword exakt einmal
- [ ] H2/H3 mit Long-Tail-Variationen
- [ ] First-Paragraph-Match für Featured Snippet (Frage stellen + 50 Wörter Antwort direkt)
- [ ] Mindestens 1 internes Link nach `/huette` und `/buchen`
- [ ] Mindestens 1 ausgehender Link auf seriöse Quelle (Wikipedia, Stadt Winterberg, Sauerlandtourismus)
- [ ] Featured Image mit aussagekräftigem Alt-Text
- [ ] Schema.org `Article` + `LodgingBusiness` Markup
- [ ] OG-Bild 1200×630 px
- [ ] URL-Slug kurz, Bindestriche, ohne Stopwörter

## Technische SEO-Hausaufgaben *(einmalig)*

- [ ] `next-sitemap` mit dynamischer Blogartikel-Liste
- [ ] structured data `LodgingBusiness` mit Lat/Lng, Preisen, Bewertungen
- [ ] `BreadcrumbList` Schema auf jeder Seite
- [ ] Local-SEO: Google Business Profile für Wiesenhütte anlegen + verifizieren
- [ ] Bing Places ebenfalls eintragen (oft übersehen, einfach gewonnen)
- [ ] Backlinks: Sauerland-Tourismus, Stadt Winterberg, ESG-Schulwebseite
- [ ] Verzeichnis-Eintrag bei gruppenhaus.de (existiert bereits, Daten aktualisieren)
- [ ] Verzeichnis-Eintrag bei Sauerland.com / langewiese.de
- [ ] PageSpeed Insights regelmäßig prüfen (Ziel: Mobile > 90)

## Was wir NICHT tun sollten

- **Keine Hochzeitsbewerbung / Junggesellenabschied** — Verein lehnt Privatfeiern ab (Quelle: Korrespondenz Dana Wolf, April 2026)
- **Keine alten Bewertungen von 2018 lassen** — wurden vom Vorstand zur Löschung freigegeben
- **Keine Tier-Fotos** — Tiere sind in der Hütte zu keiner Zeit erlaubt; Bewerbung wäre irreführend
- **Keinen Hinweis „Geburtstage feiern" oder „Mittagsruhe-frei"** — beides verletzt die Hausordnung
